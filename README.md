## 📖 这是什么？
这是一个运行在手机浏览器里的 **Web App**，它完美复刻了 90 年代老式出租车计价器的体验。
利用手机的 **GPS 定位**，它能实时计算你的行车里程和费用。

**这不是游戏，它是真的计价器。**

## ✨ 核心功能

### 1. 📟 硬核复古 UI
*   **LED 数码管**：纯正的红色辉光管字体，黑底红字，童叟无欺。
*   **物理按键**：那种按下去会“咔哒”一声的塑料质感。（这行是 AI 写的，我没听到有声音，就留着吧，警示一下人工智能写的不能完全信）
*   **翻牌器**：经典的“空车”红牌，点一下 `开始`，红牌翻下，计费开始。

### 2. 🛰️ GPS 实时计费
*   **防漂移算法**：过滤掉红绿灯停车时的 GPS 乱跳，杜绝“幽灵里程”。
*   **多地费率**：预置 **上海 / 广州 / 南京** 三地真实运价（起步价、续租、返空费）。
*   **自定义**：当然，你也可以自己设定一个“黑车费率”，比如起步价 50 元。

### 3. 💸 美式小费 Neta (核心恶搞)
*   行程结束后，你会进入一个像极了美国餐厅 POS 机的界面。
*   屏幕上会礼貌地询问：“给司机加个鸡腿？”
*   选项只有：**15% / 20% / 25%**。
*   想不给？那个 "No Tip" 按钮小到你得用显微镜找。😈

### 4. 📱 移动端适配
*   **PWA 级体验**：支持各种尺寸，布局自适应。
*   **防锁屏**：建议保持屏幕常亮，看着数字跳动是一种享受。

## 🛠️ 如何使用
1.  用手机浏览器打开网页 (推荐 Safari 或 Chrome)。
2.  允许 **获取位置权限** (必须 HTTPS)。
3.  点击 **Start**，开车出发！
4.  到达目的地，点击 **Stop** -> **Payment**。
5.  把手机递给朋友：“扫码，谢谢老板！”
